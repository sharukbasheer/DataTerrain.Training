# DataTerrain.Training

The task is to develop sample C# implementations of software design patterns, as assigned by Saravana Perumal, for internal training and learning purposes at DataTerrain.

## Purpose
- Learn design patterns with real, readable C# examples
- Use patterns in enterprise applications
- Serve as internal training material for DataTerrain

## Structure
- Creational Patterns
- Structural Patterns
- Behavioral Patterns

Each pattern includes:
- Problem statement
- Pattern explanation
- C# implementation
- Simple usage example
